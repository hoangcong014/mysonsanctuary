		1- ẩn info pannel, ẩn button group tower info, ẩn menu detail, xóa thước đo

		Lưu ý: theo thứ tự

- 3D models:
	+ Import
		++ Environment: baked texture;
		++ Add background image: background_dgm.jpg
		++ Add floor: 333333, transparent: 0.08
		++ Take thumbnail
		++ World Measure
		++ Bỏ check "Reduce Textures for Mobile" để giảm dung lượng site.
		++ Bỏ check Limit Pitch nếu muốn xoay tự do
		++ Edit rotate limit (if nescessary)
		++ Add hotspot if overview or tower group
		++ Đổi tên thành "asset + Id" hoặc "tower + Id" hoặc "workspace + Id" tương ứng (dựa vào Mã số mô tả để tìm Id trong API jsons).
